# Hoagie Hub

Hoagie Hub is a full-stack mobile application for creating and sharing Hoagies.

## Project Structure

The project is divided into two main parts:

- **[Backend](./backend)**: A NestJS API with MongoDB.
- **[Frontend](./frontend)**: A React Native mobile application built with Expo.

## Required

- [Docker](https://docs.docker.com/get-docker/) Includes compose v2 functionality
- [NodeJS] Check `.nvmrc` file for current version to install
